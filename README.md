# SpaceJava

SpaceJava è un semplice videogioco arcade in stile space shooter sviluppato in Java con Gradle.
Il progetto è stato realizzato per mettere in pratica concetti di programmazione orientata agli oggetti, modelli MVC, gestione dello stato del gioco e rendering grafico con Swing.

## Obiettivo del progetto

Il giocatore controlla una navicella e deve difendersi dagli attacchi dei nemici, sconfiggerli, accumulare punti e sbloccare nuove skin.

## Funzionalità principali

- Movimento e controllo della navicella del giocatore
- Proiettili del giocatore e dei nemici
- Nemici con diversi comportamenti e livelli di difficoltà
- Bunker protettivi
- Sistema di punteggio e record
- Shop per sbloccare e selezionare skin
- Menu iniziale, schermata game over e power-up
- Supporto alla build con Gradle e alla creazione di un jar eseguibile

## Tecnologie utilizzate

- Java
- Gradle
- Swing per l'interfaccia grafica
- JUnit 5 per i test
